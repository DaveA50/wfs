WFS_SetBlackLevelOffset

ViStatus WFS_SetBlackLevelOffset (ViSession instrumentHandle, ViInt32 blackLevelOffsetSet);

Purpose

This function sets the black offset level of the WFS camera. A higher black level will increase the intensity level of a dark camera image.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the Instrument Handle returned by the Init function to select the desired instrument driver session.
blackLevelOffsetSet ViInt32 This parameter sets the black offset value of the WFS camera. A higher black level will increase the intensity level of a dark camera image.
Valid range: 0 ... 255

Return Value

Name Type Description
status ViStatus This value shows the status code returned by the function call.
For Status Codes see function ErrorMessage.
