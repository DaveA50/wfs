WFS_GetReferencePlane

ViStatus WFS_GetReferencePlane (ViSession instrumentHandle, ViPInt32 referenceIndex);

Purpose

This function returns the Reference Plane setting of the WFS instrument.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the Instrument Handle returned by the Init function to select the desired instrument driver session.
Output
Name Type Description
referenceIndex ViInt32 (passed by reference) This parameter returns the actual Reference Plane of the WFS instrument.
Valid return values:
0   WFS_REF_INTERNAL
1   WFS_REF_USER

Return Value

Name Type Description
status ViStatus This value shows the status code returned by the function call.
For Status Codes see function ErrorMessage.
