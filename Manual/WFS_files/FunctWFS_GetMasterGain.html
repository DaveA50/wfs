WFS_GetMasterGain

ViStatus WFS_GetMasterGain (ViSession instrumentHandle, ViPReal64 masterGainAct);

Purpose

This function returns the actual set linear master gain.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the Instrument Handle returned by the Init function to select the desired instrument driver session.
Output
Name Type Description
masterGainAct ViReal64 (passed by reference) This parameter returns the actual linear master gain of the WFS camera.

Return Value

Name Type Description
status ViStatus This value shows the status code returned by the function call.
For Status Codes see function ErrorMessage.
