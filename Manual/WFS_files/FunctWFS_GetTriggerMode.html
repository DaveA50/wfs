WFS_GetTriggerMode

ViStatus WFS_GetTriggerMode (ViSession instrumentHandle, ViPInt32 triggerMode);

Purpose

This function returns the actual hardware trigger mode.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the Instrument Handle returned by the Init function to select the desired instrument driver session.
Output
Name Type Description
triggerMode ViInt32 (passed by reference) This parameter returns the actual trigger mode.
Valid trigger modes:
WFS_HW_TRIGGER_OFF  Trigger input disabled
WFS_HW_TRIGGER_HL   Trigger on high->low edge
WFS_HW_TRIGGER_LH   Trigger on low->high edge

Return Value

Name Type Description
status ViStatus This value shows the status code returned by the function call.
For Status Codes see function ErrorMessage.
