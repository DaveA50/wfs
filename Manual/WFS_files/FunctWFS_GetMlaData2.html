WFS_GetMlaData2

ViStatus WFS_GetMlaData2 (ViSession instrumentHandle, ViInt32 MLAIndex, ViChar _VI_FAR MLAName[], ViPReal64 camPitchm, ViPReal64 lensletPitchm, ViPReal64 spotOffsetX, ViPReal64 spotOffsetY, ViPReal64 lensletFm, ViPReal64 grdCorr0, ViPReal64 grdCorr45, ViPReal64 grdCorrRot, ViPReal64 grdCorrPitch);

Purpose

This function returns calibration data of the desired Microlens Array index. The number of calibrated lenslet arrays can be derived by function GetMlaCals.
Note: The calibration data are not automatically set active.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the Instrument Handle returned by the Init function to select the desired instrument driver session.
MLAIndex ViInt32 This parameter defines the index of a removable microlens array.
Valid range: 0 ... MLACount-1
Output
Name Type Description
MLAName ViChar[] This parameter returns the name of the Microlens Array.
Note: The string must contain at least WFS_BUFFER_SIZE (256) elements (char[WFS_BUFFER_SIZE]).
camPitchm ViReal64 (passed by reference) This parameter returns the camera pixel pitch in µm.
lensletPitchm ViReal64 (passed by reference) This parameter returns the Microlens Array pitch in µm.
spotOffsetX ViReal64 (passed by reference) This parameter returns the X Offset of the central MLA lenslet.
spotOffsetY ViReal64 (passed by reference) This parameter returns the Y Offset of the central MLA lenslet.
lensletFm ViReal64 (passed by reference) This parameter returns the calibrated distance (focal length) of  the Microlens Array in µm.
grdCorr0 ViReal64 (passed by reference) This parameter returns the calibrated correction value for astigmatism 0° of the Microlens Array in ppm.
grdCorr45 ViReal64 (passed by reference) This parameter returns the calibrated correction value for astigmatism 45° of the Microlens Array in ppm.
grdCorrRot ViReal64 (passed by reference) This parameter returns the calibrated correction value for rotation of the Microlens Array in 10^-3 deg.
grdCorrPitch ViReal64 (passed by reference) This parameter returns the calibrated correction value for pitch  of the Microlens Array in ppm.

Return Value

Name Type Description
status ViStatus This value shows the status code returned by the function call.
For Status Codes see function ErrorMessage.
