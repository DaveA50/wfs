WFS_GetInstrumentListLen

ViStatus WFS_GetInstrumentListLen (ViSession instrumentHandle, ViPInt32 instrumentCount);

Purpose

This function reads all Wavefront Sensor devices connected to the PC and returns the number of it.
Use function GetInstrumentListInfo to retrieve information about each WFS instrument.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter is necessary to be VXIpnp compliant.
Set the parameter to VI_NULL as this parameter is ignored.
Output
Name Type Description
instrumentCount ViInt32 (passed by reference) This parameter returns the number of WFS instruments connected to the PC.

Return Value

Name Type Description
status ViStatus This value shows the status code returned by the function call.
For Status Codes see function ErrorMessage.
