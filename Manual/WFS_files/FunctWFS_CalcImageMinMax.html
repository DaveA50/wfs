WFS_CalcImageMinMax

ViStatus WFS_CalcImageMinMax (ViSession instrumentHandle, ViPInt32 imageMin, ViPInt32 imageMax, ViPReal64 saturatedPixelsPercent);

Purpose

This function returns minimum and maximum pixel intensities in ImageBuf as well as the number of saturated pixels in percent.
Note: This function is not available in Highspeed Mode!

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the Instrument Handle returned by the Init function to select the desired instrument driver session.
Output
Name Type Description
imageMin ViInt32 (passed by reference) This parameter returns the minimum pixel intensity within ImageBuf.
imageMax ViInt32 (passed by reference) This parameter returns the maximum pixel intensity within ImageBuf.
saturatedPixelsPercent ViReal64 (passed by reference) This parameter returns the percentage of saturated pixels within ImageBuf.

Return Value

Name Type Description
status ViStatus This value shows the status code returned by the function call.
For Status Codes see function ErrorMessage.
